How to use?
1) Download the extractor.py
2) Use it like this: python extractor.py <SUBTITLE/COMMON file>
3) It will generate a candidate for subtitles and the extracted strings .txt file.

This python project takes the unpacked PSARC subtitles (you can learn how to by visiting https://github.com/rm-NoobInCoding/UnPSARC) and extracts the readable strings (most of the subtitles) from the SUBTITLES and COMMON files.
